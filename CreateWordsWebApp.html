This app is all client side.
It will generate passwords that are:

  • Secure - this actually runs everything locally - view the source now- you can see how it works.
  • Difficult to crack - there are millions of combinations here.
  • Easy to remember - these random word sequences are easy for people to remember - hard for computers to crack.
  • Good for typing on mobiles - you can add special characters for added security yourself though.
